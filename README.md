## 项目简介
本项目是一款**植物大战僵尸主题的幸存者类小游戏**，玩家操控植物自动发射子弹，抵御从右侧不断袭来的僵尸。通过击杀僵尸获取经验升级，选择技能强化自身，每5级会出现BOSS僵尸。游戏包含自动射击、扇形散射、冰冻减速、子弹穿透、护盾、无敌帧等完整机制。

---

## 功能特色
- 玩家 W/S 上下移动，自动发射扇形子弹
- 随机僵尸生成，包含普通僵尸、快速僵尸、BOSS
- 升级随机技能池：射速、子弹数、伤害、回血、护盾、穿透、冰冻
- 碰撞检测系统：距离判定 + 穿透去重
- 动态难度：血量、数量随等级提升
- 护盾、无敌帧、受伤闪烁，优化游戏平衡与体验
- 完整UI：血条、等级、经验、分数、暂停/升级界面

---

## 操作说明
- **W / S**：控制植物上下移动
- **1 / 2 / 3**：升级时选择技能
- **空格**：游戏结束后退出

---

## 核心算法
1. **僵尸移动**：按类型设置速度，向左持续移动
2. **子弹散射**：按序号计算角度，实现扇形射击
3. **碰撞检测**：两点距离判断是否击中
4. **穿透去重**：QSet 记录已击中僵尸，避免重复伤害
5. **动态难度**：僵尸数量与血量随等级动态增长
